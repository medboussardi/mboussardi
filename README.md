# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- Responsive design that works on all devices
- Modern and clean UI with Tailwind CSS
- Smooth scrolling navigation
- Mobile-friendly navigation menu
- Contact form
- Project showcase
- Skills and experience sections
- Social media integration

## Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the Tailwind CSS build process:
```bash
npm run build
```

4. Open `index.html` in your browser to view the website.

## Development

The website uses Tailwind CSS for styling. To make changes to the styles:

1. Edit the `src/input.css` file
2. The changes will be automatically compiled to `dist/output.css`

## Deployment to GitHub Pages

1. Push your code to GitHub:
```bash
git add .
git commit -m "Initial commit"
git push origin main
```

2. Go to your repository settings on GitHub
3. Navigate to "Pages" in the sidebar
4. Under "Source", select "main" branch
5. Click "Save"

Your website will be available at `https://yourusername.github.io/portfolio`

## Customization

- Edit `index.html` to update content
- Modify `tailwind.config.js` to customize colors and theme
- Update `src/input.css` for custom styles
- Edit `js/main.js` for custom JavaScript functionality

## Technologies Used
